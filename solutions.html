Solutions

Delfos IM

Delfos is an intelligent maintenance platform that uses state of the art machine learning modules, such as Google Brain’s Tensor Flow and IBM Watson to integrate all the data already available from the wind turbines and transform it into meaningful information. That information is used to create a failure database with failure prediction models and failure solution roadmaps.

Performance Analysis

Delfos IM uses Machine Learning to identify the best performance indicators and create the most efficient KPI. Underperformance can be identified in advance, avoiding energy loss.

Predicive Maintenance

Delfos IM uses the data from the wind turbine to identify the best prediction model for each type of failure and associate it with the best maintenance practices. Our flexible platform uses more than 50 different prediction models from different industries to find the best fit for each normal behavior and failure.
